Text prefixer tool What is a text prefixer?
With this tool you can prepend a prefix at the beginning of text. You can prepend words, letters, or numbers. Anything that you specify in prefix option will get affixed to text. It has three modes of operation â€“ single-prefix function that will add prefix text only once at the very beginning, line-by-line function that will affix every text line, and paragraph function that will add text to paragraphs only (a paragraph is one or more text lines followed by two new lines.) There"s also a skip-empty-lines function, which when selected will avoid adding prefix to blank text lines. Textabulous!
